# Waste Collection Point Database

This directory contains the scripts needed to manage the database and related resources for waste collection points used in the Trash Optimizer project.

## Data sources

- Nantes Pays de la Loire Open Data: [hhttps://data.nantesmetropole.fr](https://data.nantesmetropole.fr/api/explore/v2.1/catalog/datasets)
- Le Relais: [https://www.lerelais.org](https://www.lerelais.org/)
- Ecosysteme: [https://www.ecosysteme-reemploi.fr](https://www.ecosysteme-reemploi.fr/)
